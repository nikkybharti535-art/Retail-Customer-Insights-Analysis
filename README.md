# 🛍️ Retail Customer Insights Analysis

## 📖 About the Project

Retail Customer Insights Analysis is a data analytics project that explores customer shopping behavior using transactional data. The project focuses on cleaning the data, performing exploratory data analysis (EDA), and answering business-related questions using SQL to generate meaningful insights.

This project was completed as part of my learning journey in Data Analytics to strengthen my skills in Python, SQL, data cleaning, and business analysis.

---

## 📊 Dataset Information

- **Total Records:** 3,900
- **Total Features:** 18

The dataset includes:

- Customer Demographics (Age, Gender, Location, Subscription Status)
- Purchase Details (Item Purchased, Category, Purchase Amount, Season, Size, Color)
- Shopping Behavior (Discount Applied, Previous Purchases, Frequency of Purchases, Review Rating, Shipping Type)

### Data Cleaning

- Checked dataset structure and summary statistics.
- Identified missing values in the **Review Rating** column.
- Filled missing values using the median rating of each product category.
- Renamed column names using snake_case.
- Created new features such as **Age Group** and **Purchase Frequency**.
- Removed redundant columns after checking data consistency.
- Loaded the cleaned dataset into PostgreSQL for SQL analysis.
